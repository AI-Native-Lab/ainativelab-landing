# AI Native Lab — Landing page (linkhub)

Trang linkhub chính thức của AI Native Lab.

- **Live:** https://ainativelab.org
- **Hosting:** Netlify (site `ainativelab`)
- **Domain/DNS:** Cloudflare

## Cấu trúc

Chỉ gồm 1 file tĩnh duy nhất:

- `index.html` — toàn bộ trang (HTML + CSS inline, font Be Vietnam Pro). Không cần build, không phụ thuộc thư viện ngoài.

## Cập nhật nội dung

1. Sửa `index.html`.
2. Commit / push lên nhánh `main`.
3. Nếu repo đã nối với Netlify thì trang tự deploy lại. Nếu chưa nối, deploy thủ công lên site Netlify `ainativelab`.
